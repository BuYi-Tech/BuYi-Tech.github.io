一、GIt与github绑定

获取ssh密钥

大家之前都已经把git下载好了吧，今天便是上一步的结合，将GitHub和git绑定起来。

首先打开git，依旧是右键点git bash然后输入ssh，弹出下图所示便是已经安装了SSH（安全外壳协议，Secure Shell 的缩写）

然后，输入 ssh-keygen -t rsa 命令（注意空格），表示我们指定 RSA 算法生成密钥，然后敲四次回车键，之后就就会生成两个文件，分别为秘钥 id_rsa 和公钥 id_rsa.pub. 文件的位置在 Git Bash 上面都有显示。

绑定github

打开GitHub网站 如图所示操作

将复制的公钥 id_rsa.pub 的内容粘贴到 key 内，tittle可不填

验证是否成功，我们可以通过在 Git Bash 中输入 ssh -T git@github.com进行检验

出现如图所示语句为绑定成功，第一次绑定可能还需要输入yes

然后我们在本地任意创建一个文件夹，在其中右击呼出git bash在里面输入git clone+“你刚才复制的东西”

在你新建的文件夹里出现下图所示即为操作成功

然后我们新建一个测试文件

然后从此目录进入 Git Bash，输入 git status 命令查看仓库状态

现在我们使用 git add 命令将文件添加到了「临时缓冲区」，再用 git commit -m “提交信息” 将其提交到本地仓库，如下图

如果你是第一次提交的话，就会出现下图的情况，会让你输入用户名和邮箱

按照它提示的语句输入即可

接下来再输入git push origin main命令将其提交到GitHub就好了
